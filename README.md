# Análise Empírica de Complexidade de Algoritmos

Trabalho prático desenvolvido para a disciplina de Estruturas de Dados Básicos II.

## 🎯 Objetivo
Este projeto implementa um programa em C++ para avaliar empiricamente o tempo de execução de dois algoritmos de ordenação:
- **Insertion Sort** ($O(n^2)$)
- **Merge Sort** ($O(n \log n)$)

O objetivo é coletar os tempos de execução para diferentes tamanhos de entrada ($N$) e comparar o crescimento do tempo com as funções assintóticas teóricas.

## 🚀 Como executar
O arquivo `main.cpp` contém todo o código-fonte. Ele pode ser compilado localmente utilizando qualquer compilador C++ (como o `g++`) ou executado diretamente em ambientes de desenvolvimento online, como o Replit. 
O programa gerará automaticamente um arquivo `.csv` com os resultados das medições de tempo.
